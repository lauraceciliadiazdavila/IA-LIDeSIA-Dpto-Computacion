# Procesamiento de Datos e Imágenes

Esta carpeta contiene los scripts y notebooks utilizados para el preprocesamiento de los datos.

Las tareas realizadas incluyen:

- Limpieza del dataset
- Depuración de imágenes
- Entrenamiento de arquitecturas CNN y ViT

Se incluyen los siguientes archivos:
 - Notebook del proceso de depuración de cada dataset individual
 - Notebook de la depuración y creación de carpetas de imágenes
 - Código arquitectura CNN
 - Código arquitectura ViT
 - Resultados .txt del entrenamiento 1 de ViT
 - Resultados .txt del fine tunning del modelo 1 de ViT
